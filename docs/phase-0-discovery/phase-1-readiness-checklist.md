# Phase 1 Readiness Checklist

Use this checklist before starting drone inspection implementation.

## Business Readiness

- [ ] Pilot zone selected.
- [ ] Pilot objective approved.
- [ ] Baseline KPIs collected.
- [ ] Success criteria documented.
- [ ] Manual validation team identified.
- [ ] Operating hours and flight windows approved.

## Warehouse Readiness

- [ ] Rack/bin location master collected.
- [ ] Pilot zone mapped physically.
- [ ] No-fly zones mapped.
- [ ] Flight corridors drafted.
- [ ] Rack labels verified.
- [ ] Lighting issues documented.
- [ ] Wi-Fi/network quality checked.

## Data Readiness

- [ ] SKU master access confirmed.
- [ ] WMS location data access confirmed.
- [ ] Cloud SQL schema approved.
- [ ] Object storage bucket plan approved.
- [ ] Dataset label taxonomy approved.
- [ ] Annotation workflow defined.
- [ ] Retention policy confirmed.

## Technical Readiness

- [ ] Drone hardware candidate selected.
- [ ] Camera requirements defined.
- [ ] Edge upload approach defined.
- [ ] Cloud environment selected.
- [ ] Kubernetes/deployment direction approved.
- [ ] MLflow model registry direction approved.
- [ ] API integration path documented.

## Safety Readiness

- [ ] Safety owner sign-off received.
- [ ] Emergency stop process defined.
- [ ] Manual override process defined.
- [ ] Incident logging process defined.
- [ ] Controlled flight windows approved.
- [ ] Worker/forklift separation approach documented.

## Approval Gate

Phase 1 should not start until the following owners approve:

- [ ] Warehouse operations.
- [ ] Inventory control.
- [ ] Safety/compliance.
- [ ] IT/cloud architecture.
- [ ] Security.
- [ ] WMS/ERP owner.
- [ ] Business sponsor.
